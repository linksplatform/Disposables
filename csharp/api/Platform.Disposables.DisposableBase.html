    Class DisposableBase

    Represents a base implementation for IDisposable interface with the basic logic necessary to increase the likelihood of correct unmanaged resources release.

    Представляет базовую реализацию для интерфейса IDisposable с основной логикой необходимой для повышения вероятности корректного высвобождения неуправляемых ресурсов.
